# AI Name Generator

Static English name generator tool site.

Live: https://baiyeyun.xyz/

## Tech Stack

- HTML / CSS / JavaScript
- Cloudflare Pages
- GitHub

## Tools (30 pages)

- Creator: YouTube, TikTok, Instagram, Discord, Podcast, Blog, Band, Twitch, Twitter
- Business: Business, Brand, Shop, Bakery, Clothing Brand, Etsy Shop, Restaurant, Coffee Shop, Photography Business, Hair Salon, Startup
- Pets: Pet, Cat, Dog
- Gaming: Gamer Tag, Fantasy, DnD, Guild, Team, Dragon
- Fun: WiFi
