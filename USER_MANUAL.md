# 使用说明书

## 这个网站是什么

这是一个英文 `AI Name Generator` 工具站。

用户可以用它生成：

- 商业名称
- 品牌名称
- YouTube 名称
- TikTok 用户名
- Instagram 用户名
- 宠物名字
- 游戏名
- 奇幻角色名
- WiFi 名称
- Etsy 店铺名

当前域名计划使用：

- `baiyeyun.xyz`

## 当前网站入口

本地预览地址：

- `http://127.0.0.1:4173/index.html`

当前还没有正式发布到公网。

## 网站已经有什么功能

### 1. 首页

首页包含：

- 网站定位
- 热门工具入口
- 分类入口
- 使用流程说明

### 2. 工具总页

地址：

- `/tools/index.html`

功能：

- 查看全部工具
- 按分类浏览
- 搜索工具

### 3. 单个工具页

每个工具页都有：

- 名字生成器
- 关键词输入框
- 风格选择
- 一键生成
- 单个名字复制
- 全部结果复制
- 保存喜欢的名字
- 候选名单复制
- 候选名单清空
- 使用技巧
- FAQ
- 相关工具
- 继续浏览推荐
