      Checklist

      A quick naming checklist

      • Try multiple combinations before settling on one.
      • Make sure both people like the name.
      • Check that it does not sound like an existing word with a bad meaning.
      • Test it out loud — does it sound natural?

      FAQ

      Couple Name Generator FAQ

      How do you make a couple name?

      Take parts of each name and blend them. Common methods: first half of one name + second half of the other, or first syllable of each name.

      What is a ship name?

      A ship name (from "relationship") is a blended name for a couple, often used in fandoms for fictional or celebrity couples.
